# ASE-Lab. 衛星データ解析チーム SCALE UP
ASE-Lab. 衛星データ解析チーム SCALE UPは、衛星データを利活用したい、興味がある大学生を中心に衛星データ解析についてアウトリーチするチームです。
衛星データ解析に関するイベントの開催や、衛星データ解析に関する情報発信を行っています。
また、衛星データ解析に関するプロジェクトを実施しています。
衛星データ解析に関するプロジェクトは、衛星データ解析に関する知識や技術を身につけることができる内容となっています。

# リポジトリ構成
- `data/`: 衛星データ解析に関するデータセットを格納するディレクトリ
- `notebooks/`: 衛星データ解析に関するノートブックを格納するディレクトリ
- `scripts/`: 衛星データ解析に関するスクリプトを格納するディレクトリ
- `README.md`: リポジトリの説明を記載するファイル
- `LICENSE`: ライセンスを記載するファイル
